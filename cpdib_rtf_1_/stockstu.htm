StockStu

설명

종목 코드/업종 코드에 대해서 분,틱 차트 위한 데이터를 구분,주기,count ,시간외 데이터  구분으로서 요청하고 수신한다.

StockStu 대신 StockChart의 사용을 권장합니다.

Method

object.SetInputValue(type,value)

type에 해당하는 입력 데이터를 value 값으로 지정합니다

  • type: 입력 데이터 종류

0 - (string) 종목 코드/업종 코드

1 - (char) 구분

코드

내용

'T'

Tick데이터

'M'

분 데이터

2 - (short) 주기

- Tick mode 일 때 최대 120

- 분 mode 일 때 최대 60

3 - (short) count

- 요청 데이터 수는 최대 약 1700

4 - (char) 장전시간외 구분

코드

내용

'1'

시작봉과 마지막 봉에 시간외 거래량 합산
[Default]

'2'

첫봉에 시간외 거래량 제거

'3'

시작봉과 마지막 봉 모두 시간외 거래량 제거

'4'

마지막 봉에 시간외 거래량 제거

  • value: 새로 지정할 값
