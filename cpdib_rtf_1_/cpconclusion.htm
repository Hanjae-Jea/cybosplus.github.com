CpConclusion

대신증권 차세대 오픈으로 변경된 오브젝트입니다.

변경된 내용은 [CpTrade_GIANT]에서 꼭 확인하셔서 이용에 착오가 없으시기 바랍니다.

 

설명: 주문한 것에 대한 체결 내역을 요청하고 수신합니다

관련 CYBOS 대표화면: CYBOS 주식체결 티커바

통신종류: Subscribe/Unsubscribe

모듈위치: CpDib.dll

Method

object.SetInputValue(type,value)

사용하지 않음

value = object.GetHeaderValue(type)

type에 해당하는 헤더 데이터를 반환합니다

  • type: 데이터 종류

0 - (string) 주문 구분 내용

1 - (string)계좌명

2 - (string)종목명

3 - (long)체결수량

4 - (long)체결가격

5 - (long)주문번호

6 - (long원주문번호

7 - (string)종합계좌번호

8 - (string)계좌구분

"10":위탁자 "29":공동계좌 "61":일반저축 "63":장기저축 "64":장기주택저축 "65":세금우대저축 "66":주식저축 "68":신주식저축 "69":근로자우대

9 - (string)종목코드

10 - (string)체결구분("1": 체결, "2": 확인, "3": 거부, "0": 접수)

11 - (string)신용구분

"01","11": 유통융자 "03","33":자기융자 "05","55":유통대주 "07","77":자기대주 "08","88":매입대출 "99":담보대출 "98":플러스론 Else:현금

12 - (string)매매구분( 체결 구분에서 1:체결일 때만 1:매도, 2:매수)
13 - (string)주문유형 ( 체결 구분에서 2:확인일 때만 1:정정, 2: 취소)

  • 반환 값: 데이터 종류에 해당하는 값

value = object.GetDataValue (Type,index)

사용하지 않음

object.Subscribe()

주문 체결 data 수신을 신청한다

object.Unsubscribe()

주문 체결 data 수신을 해지한다

object.Request()

사용하지 않음

object.BlockRequest()

사용하지 않음

Event

Object.Received

주문 체결 데이터(매수 주문,매도 주문,정정 주문,취소 주문)를 수신했을 때 발생하는 이벤트

<주의>

복수 계좌를 가지신 분들은 U-CYBOS 로 사인 온 된 복수계좌에 한해서만 체결 데이터를 받으실 수 있습니다
