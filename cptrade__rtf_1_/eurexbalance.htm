EurexBalance

 

설명: EUREX 실시간 계좌평가 손익 데이터를 요청하고 수신한다.

통신종류: Request/Reply

모듈위치: CpTrade.dll

Method

object.SetInputValue(type,value)

type에 해당하는 입력 데이터를 value 값으로 지정합니다

  • type: 입력 데이터 유무

0 – (string) 계좌번호

1 - (string) 상품관리구분[Default:55]

2 - (string) 종목코드

3 - (long) 요청개수[Default:20]-최대 50

  • value: 새로 지정할 값

value = object.GetHeaderValue(type)

type에 해당하는 헤더 데이터를 반환합니다

  • type: 데이터 종류

0 - (long) 수신개수

  • 반환값: 데이터 종류에 해당하는 값

value = object.GetDataValue (Type,index)

type에 해당하는 데이터를 반환합니다

  • type: 데이터 종류

0 - (string) 종목코드

1 - (string) 종목명

2 - (string) 잔고구분

3 - (long) 잔고수량

4 - (double) 평균단가

5 - (long) 청산가능수량

6 - (long) 매매/체결수량

  • 반환값: 데이터 종류의 index번째 data
