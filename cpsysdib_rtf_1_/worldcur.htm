Method

object.SetInputValue(type,value)

type에 해당하는 입력 데이터를 value 값으로 지정합니다

  • type: 입력 데이터 종류

0 - (string) 종목코드 - 해외지수 및 환율 심볼코드
 [해외지수코드 list는 CpUtil의 CpUsCode 참조]

  • value: 새로 지정할 값

value = object.GetHeaderValue(type)

type에 해당하는 헤더 데이터를 반환합니다

  • type: 데이터 종류

0 - (string) 종목코드

1 - (float) 현재가

2 - (float) 전일대비

3 - (float) 전일대비율

4 - (ulong) 거래량

5 - (float) 시가

6 - (float) 고가

7 - (float) 저가

8 - (float) 전일종가

9 - (float) 52주최고가

10 - (float) 52주최저가

11 - (long) 현지거래시간(초:단위는 분이지만 호환성을 위해 00을 붙여 초단위로 제공)

12 - (long) 현지거래일자 - yyyymmdd

  • 반환값: 데이터 종류에 해당하는 값
