Method

object.SetInputValue(type,value)

type에 해당하는 입력 데이터를 value 값으로 지정합니다

  • type: 입력 데이터 종류

0 - (string)종목코드
1 - (usort)요청개수

  • value: 새로 지정할 값

value = object.GetHeaderValue(type)

type에 해당하는 헤더 데이터를 반환합니다

0 - (string)종목코드
1 - (ushort)수신개수
2 - (ulong)누적매도체결수량
3 - (ulong)누적매수체결수량

  • 반환값: 데이터 종류에 해당하는 값

value = object.GetDataValue(Type,Index)

  • type: 데이터 종류

0 - (ulong)시간

5 - (ulong)누적거래량
