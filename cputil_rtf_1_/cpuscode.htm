CpUsCode

설명: 해외종목 코드 조회

모듈위치: CpUtil.dll

Method

Value = object.GetUsCodeList(USTYPE UsType)

UsType에 해당하는 해외종목 코드 목록을 얻는 함수

  • UsType(USTYPE): 해외종목 종류

USTYPE_ALL : 전종목 - 1

USTYPE_CONTRY : 국가 대표 - 2

USTYPE_UPJONG : 업종 3

USTYPE_JONGMOK : 종목 - 4

USTYPE_DR : 예탁증서 - 5

USTYPE_RAW : 상품선물 - 6

USTYPE_EXCHANGE : 환율 - 7

  • 반환값(String Array): 해당 해외종목 종류 코드 목록

value = object.GetNameByUsCode(String UsCode)

입력한 해외종목 코드의 이름을 얻는 함수

  • UsCode(String) : 이름을 요청하는 해외종목코드

  • 반환값(String): 해당 해외종목 코드의 이름

 

VB ex)

Dim CpUsCode As New CpUsCode

Dim codes As Variant

codes = CpUsCode.GetUsCodeList(USTYPE_CONTRY)

For i = LBound(codes) To UBound(codes)

Debug.Print CpUsCode.GetNameByUsCode(codes(i))
