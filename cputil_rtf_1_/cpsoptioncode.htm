CpSOptionCode

설명: CYBOS에서 사용되는 개별 주식 옵션 코드 조회 작업을 함.

모듈위치: CpUtil.dll

Method

Value = object.CodeToName (code)

Code에 해당하는 옵션 종목명을 반환한다

  • code: 개별 주식 옵션코드

  • 반환값: 개별 주식 옵션 종목명

value = object.GetCount()

옵션코드 수를 반환한다

  • 반환값: 개별 주식 옵션코드 개수

value = object.GetData(type,index)

해당 인덱스의 옵션종목 데이터를 구한다

  • type : 데이터 종류

0 - 개별 주식 옵션코드

1 - 개별 주식 옵션명

2 - 콜/풋 구분

3 - 행사월

4 - 행사가

  • index : 코드 인덱스

  • 반환값: 해당 데이터
