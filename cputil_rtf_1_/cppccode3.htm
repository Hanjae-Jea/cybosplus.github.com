CpPcCode3

설명:  CYBOS에서 사용되는 각종 코드리스트 및 컨버트 작업을 함.

 삭제된 오브젝트입니다. CpCodeMgr로 대체 사용하세요.

모듈위치: CpUtil.dll

공통

tno ( Table Number ) : 코드 테이블 번호

0 - 지점/부서 코드  (CP에서 사용되지 않아서 삭제되었습니다)

1 - 상품 코드   (CP에서 사용되지 않아서 삭제되었습니다)

2 - 지역 코드   (CP에서 사용되지 않아서 삭제되었습니다)

3 - 영업지역 코드  (CP에서 사용되지 않아서 삭제되었습니다)

4 - 그룹(계열사) 코드  (CpCodeMgr.GetGroupList 로 대체하세요)

5 - 지방 코드   (CP에서 사용되지 않아서 삭제되었습니다)

6 - 대신 업종코드 (CP에서 사용되지 않아서 삭제되었습니다)

7 - 증권전산 업종코드 (CpCodeMgr.GetIndustryList 로 대체하세요)

8 - DERI 코드  (CP에서 사용되지 않아서 삭제되었습니다)

9 - 지점 코드 2  (CP에서 사용되지 않아서 삭제되었습니다)

10 - 지역 코드 2  (CP에서 사용되지 않아서 삭제되었습니다)

11 - 거래원 코드  (CpCodeMgr.GetMemberList로 대체하세요)

12 - 장외업종코드1  (CpCodeMgr. GetKosdaqIndustry1List로 대체하세요)

13 - 장외업종코드2  (CpCodeMgr.GetKosdaqIndustry2List로 대체하세요)

14 - 국가코드  (CpUsCode로 대체하세요)

 

Method

Value = object..CodeToName(tno,code)

tno에 해당하는 테이블에서 code에 해당하는 문자열을 반환합니다

  • tno : 코드 테이블 번호

  • code: 코드값

  • 반환값: 코드에 해당하는 문자열

value = object.GetCount(tno)

코드테이블의 항목 개수를 반환합니다

  • tno: 코드 테이블 번호

  • value: 코드 테이블의 항목 수

value = object.GetCode(tno,index)

코드테이블의 index 번째 항목의 코드를 반환합니다

  • tno: 코드 테이블 번호

  • index: 레코드 번호

  • value: 코드

value = object.GetName(tno,index)

코드테이블의 index 번째 항목의 문자열을 반환합니다

  • tno: 코드 테이블 번호

  • index: 레코드 번호

  • value: 문자열
